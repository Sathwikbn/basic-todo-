# To-Do List Web App

A simple, dynamic To-Do list application built with vanilla JavaScript, HTML, and CSS. Users can add, complete, and remove tasks with instant UI updates.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Remove tasks
- ✅ Clean, responsive design
- ✅ Instant UI updates (no page reload)
- ✅ Keyboard support (Enter to add tasks)

## Getting Started

### Prerequisites
- Web browser (Chrome, Firefox, Safari, etc.)
- VS Code (recommended)
- Live Server extension (for development)

### Running the App

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sathwikbn/basic-todo-.git
   cd basic-todo-
   ```

2. **Open with Live Server:**
   - Open the project folder in VS Code
   - Right-click on `index.html`
   - Select "Open with Live Server"

3. **Or simply open in browser:**
   - Double-click `index.html` to open in your default browser

## How to Use

1. **Add a Task:** Type your task in the input field and click "Add" or press Enter
2. **Complete a Task:** Click on the task text to toggle completion status
3. **Remove a Task:** Click the "Remove" button next to any task

## Technical Implementation

- **Vanilla JavaScript:** No frameworks or libraries used
- **Event Listeners:** Handles user interactions (clicks, keyboard input)
- **DOM Manipulation:** Dynamically creates and modifies HTML elements
- **CSS Flexbox:** Responsive layout design
- **Local State:** Tasks are stored in memory (resets on page reload)
